Run Concept Definitions
Zone
Inside Zone, Outside Zone, Sweep, Stretch, Duo, Off Tackle, Wide Zone, Slash, QB Zone, Read Option, Speed Option, Midline, Any Jet Motion Zone Run
Power
Dive, Slam, Power O, Dart, Base, Iso, Lead, Blast, Punch, Buck, Smash, QB Power, Power Option, Any Jet Motion Power Run
Counter
Counter, GT Counter, H/G Counter, QB Wrap, QB Counter, Veer, Triple Option, Slot Option, Any Jet Motion Counter Run
Trap
01 Trap, 56 Trap, Trap, Wham, Wide Trap, Pin & Pull Toss, Crack Toss, Toss, Zone Toss, QB Trap

HB Draw & HB Direct may be placed in any concept. HB Direct counts as a trick play.

Audible Requirements

Every formation/set must have: 1 Quick Pass · 1 Run · 1 Deep Pass · 1 Play-Action
RPOs, Option Plays, and Trick Plays may NOT be assigned as audibles.

Creation Protocol

You must stream the full creation of your playbook. Start by removing all non-goal-line plays. When finished, do a full play-by-play review, save as SFL_YourTeamName, then end stream.
